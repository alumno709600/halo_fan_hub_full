# Halo Fan Hub (Vite + Sass) - Full Project

Contenido incluido:
- index.html + páginas: /pages/games.html /pages/books.html /pages/series.html
- styles/ (Sass partials y main.scss)
- main.js con navegación y HUD scan demo
- assets/ (tus imágenes subidas como hero.jpg, cover1.jpg, screenshot1.png)
- vite.config.js y package.json

Instrucciones rápidas:
1. `npm install`
2. `npm run dev`
3. Abre http://localhost:5173

Producción:
- `npm run build` y `npm run preview`.

¡Listo! Puedes editar `styles/` y las páginas en `/pages`.
